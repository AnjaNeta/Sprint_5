# UI автотесты для сервиса «Доска»

## Описание проекта
Проект содержит UI автотесты для учебного сервиса «Доска» (https://qa-desk.stand.praktikum-services.ru/).

## Структура проекта:
- locators.py - локаторы элементов
- conftest.py - фикстуры
- tests/ - директория с тестами

test_registration.py - тесты регистрации
test_login.py - тесты авторизации
test_logout.py - тесты выхода
test_ads.py - тесты объявлений
- README.md
- .gitignore


## Тестируемый функционал
1. Регистрация пользователя
2. Регистрация пользователя c email не по маске  *******@*******.***
3. Регистрация уже существующего пользователя
4. Авторизация пользователя
5. Выход пользователя
6. Создание объявления неавторизованным пользователем
7. Создание объявления авторизованным пользователем

## Требования
- Python 3.11+
- Google Chrome
- ChromeDriver

## Запуск тестов
pytest tests/ -v
